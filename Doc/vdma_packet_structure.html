3. Line Data (Row) — Strict In-Order

1 Line = Header Prefix + (active_chips × stops_per_chip) cells. Chip 순서: 항상 chip0 → chip1 → chip2 → chip3.

Total data: 4 chips × 32 beats = 128 beats × 8B = 1024B per row

Total line: 48B (header) + 1024B (data) = 1072B = 134 beats @64-bit
